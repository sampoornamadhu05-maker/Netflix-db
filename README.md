# Netflix Movies vs TV Shows Analysis (2025)

- Project Overview

This project analyzes Netflix content data up to 2025 using Python.
It compares the total number of Movies and TV Shows and visualizes the results using a bar chart.

The goal of this project is to practice:

Data analysis using Pandas
Data visualization using Matplotlib
Working with CSV datasets
Basic exploratory data analysis (EDA)

- Technologies Used
Python
Pandas
Matplotlib
VS Code

- Datasets Used
netflix_movies_detailed_up_to_2025.csv
netflix_tv_shows_detailed_up_to_2025.csv

These datasets contain detailed information about Netflix Movies and TV Shows including:

Title
Country
Type
Other metadata

- What This Project Does

✔ Loads both datasets using Pandas
✔ Displays total number of Movies and TV Shows
✔ Shows Top 5 Countries by content count
✔ Creates a bar chart comparing Movies vs TV Shows

- Output Visualization

The project generates a bar chart:

Movies vs TV Shows:
X-axis → Type (Movies, TV Shows)
Y-axis → Count
Shows comparison of total content available
